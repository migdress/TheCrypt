ABOUT

The Crypt is a simple handy tool oriented for students to learn and understand the cryptography by usage

The Crypt is developed and maintained by Miguel Morcillo ( Linkedin ). If you want to contribute contact me --> miguelmorcillo@unicauca.edu.co.

The Crypt is open source, you can download the source here --> GitHub and feel free to extend, modify, cite or anything under the terms of GNU/General Public Licence

The Crypt is built with plain HTML5, CSS and Javascript and also uses AJAX to load the content dynamically

The Crypt is written in english because as a developer I want to reach people around the world and also because I want to encourage students to get used as they will have to as professionals.

The Crypt is provided AS IS and NO WARRANTY should be assumed, It's a learning application and should be used for learning purposes

Special thanks to MSc. Siler Amador Donado by sharing his knowledge in cryptography, informatic security and computer forensics.
