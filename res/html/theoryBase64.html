How does work?:

Step 1:
A character is transformed to its numeric value according to the charset, normally UTF-8, originally ASCII, for example the character 'M' has the value '77' in ASCII

Step 2:
The numeric value (which is base 10) is converted to base2 or binary, conforming groups of 8 bits (It's important to understand that there have to be 8 bits, adding zeros ahead if needed), for example the numeric value '77' is represented with the 8-bit binary value '01001101'.

Step 3:
The total length of the 8-bit binary stream needs to be an exact multiple of 24, so for example for the 8-bit binary value '01001101' to be encoded to base64, 16 bits more need to be added resulting a binary string like this '010011010000000000000000', then the 24 bit stream is divided into groups of 6 bits (because 26=64), resulting something like this '010011''010000' '000000'000000', those values are represented in base 10 as '19''16''0''0', resulting something like this ' (Because 2*2*2*2*2*2=64, that's actually which is named base64) to represent a value of the alphabet.
